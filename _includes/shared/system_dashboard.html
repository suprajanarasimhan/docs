Note: System dashboards are read-only. If you want to make your own version of a system dashboard, clone the system dashboard, and make changes to the clone. If you have a dashboard with the same URL as a system dashboard, we save the dashboard with "-clone" added to the URL and "(Cloned)" added to the name.

On the All Dashboards page, system dashboards are indicated with the Tanzu Observability by Wavefront logo icon () next to the dashboard name.
